The anti-jamming system works thanks to the nodes and the satellite:
When a node or satellite detects too much junk traffic, it asks other nodes if they also hear that junk or if the node is simply in a "busy" area. If multiple nodes hear junk, 
they send a global packet asking the user to change the communication frequency. To change, simply press
"OK" on the screen or keypad. The LoRa module automatically changes frequency and the G-Talk devices resume normal operation.
